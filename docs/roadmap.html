Owner pre-work (cert procurement — start now)

Stage 21 (distribution polish) is gated on Apple Developer ID cert. Multi-day lead time. None of this needs a coding agent — Geoff can kick it off in parallel with everything else.

  • Enroll in the Apple Developer Program ($99/year individual or $299/year organization). 1–2 business days verification; week+ for orgs. Start matters more than completion. → developer.apple.com/programs
  • Register the bundle ID (e.g. com.geoffreydudgeon.duo) in App Store Connect.
  • Generate a "Developer ID Application" certificate via Keychain Access + the Apple Developer portal. Signs the .app.
  • Generate an App Store Connect API key for notarytool. Preferred over app-specific password.
  • Capture the Team ID from the Developer portal — Stage 21 wires it into electron-builder.yml.

When all five are done, hand the agent: cert name as it appears in Keychain, Team ID, API key file path + key ID + issuer ID. Stage 21 picks up from there. Existing dev cert is expired per Team C's audit (2026-04-26).
